Handwriting Fonts

Download the Victorian Modern Cursive Script font

Windows - To install the regular and bold Victorian Modern Cursive fonts:

Download .ZIP for Windows

1: Open the ZIP file.

2: Open each file and click "Install"

3: You will then be able to select the fonts from your word processing application.

Mac - To install the regular and bold Victorian Modern Cursive fonts:

Download the Victorian Modern Cursive Script font

Download .ZIP for Mac!

1: Download and open the ZIP file. A folder VICMODERNCURSIVE will be created.

2: Go to the 'Applications' folder run 'Font Book'

- Menu option: File, Add Fonts...

- Go to folder VICMODERNCURSIVE

- Select font VICMODERNCURSIVEReg.bmap, and click Open

3: You will then be able to select the fonts from your word processing application.
